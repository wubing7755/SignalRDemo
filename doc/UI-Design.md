## 3. 页面组件结构

```
Client/Pages/
├── Index.razor           # 首页 (落地页)
│   ├── Hero.razor        # 主视觉区
│   ├── Features.razor    # 功能特性
│   └── Footer.razor      # 页脚
│
├── ChatRoom.razor        # 聊天室主页
│   ├── RoomList.razor    # 房间列表 (左侧)
│   ├── MessageArea.razor # 消息区域 (中部)
│   ├── UserList.razor   # 用户列表 (右侧)
│   └── InputArea.razor  # 输入区域 (底部)
│
└── Settings.razor       # 用户设置 (弹窗)
```

---

## 4. 实现优先级

| 优先级 | 页面/组件 | 描述 |
|--------|----------|------|
| P0 | Index.razor | 首页 (简洁落地页) |
| P0 | ChatRoom.razor | 聊天室主框架 |
| P1 | RoomList.razor | 左侧房间列表 |
| P1 | UserList.razor | 右侧用户列表 |
| P2 | Settings.razor | 设置弹窗 |
